# Data assets — Volume VI, Chapter 2 (Metabolism and bioenergetics)

## Files

| File | Used by |
|---|---|
| `standard_dg_cellular_reactions.csv` | Section 2.1 (ATP currency), Section 2.5 (free-energy balances), the calculation exercises on $\Delta G$. Standard Gibbs free energies for ATP / GTP / phosphagen hydrolysis and for key glycolytic and TCA reactions, sourced from Nelson and Cox (Lehninger 7e). |
| `atp_yield_per_substrate.csv` | Section 2.2 (catabolism), figure `fig-fuel-yield`, the calculation exercise on palmitate vs glucose yield. Per-molecule and per-gram ATP yields for the main cellular fuels under their canonical catabolic pathway. |
| `respiratory_quotient_by_fuel.csv` | Section 2.2 (oxygen consumption), the worked example on muscle contraction. Respiratory quotient (RQ = $\text{CO}_2$ produced / $\text{O}_2$ consumed) by fuel type; used to infer substrate from indirect calorimetry. |
| `basal_metabolic_rate_by_organ.csv` | Section 2.6 (worked examples), the chapter project (per-organ method). Organ masses, specific metabolic rates, and percent contribution to basal metabolic rate; consistent with the Elia (1992) compilation cited downstream of textbook physiology references. |
| `photosynthesis_efficiency_by_crop.csv` | Section 2.3 (photosynthesis), figure `fig-photo-z-scheme` caption, the estimation exercise on algal photobioreactor productivity. Typical and record field efficiencies for crops and algae, with a photovoltaic comparison line. |

## Provenance

All values are taken from standard references cited in the chapter
text (Nelson and Cox 2017, Lehninger 7e; Stryer 2015; Vander Heiden
2009; Zhu 2010 on photosynthesis efficiency). Where ranges appear in
the source, the table records a representative value and the
source-note column flags the source. No experimental data, no
clinical data, no PHI.
