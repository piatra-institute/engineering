\caption[Welded plate girder: cross-section and stiffened web]{A welded
plate girder. The cross-section (left) is built up from two flange
plates and a web plate joined by longitudinal fillet welds, so the
designer chooses the depth, the flange area, and the web thickness
independently rather than accepting the proportions of a rolled shape.
The web is deliberately slender, which makes it economical in bending
but prone to shear buckling, so the elevation (right) carries transverse
intermediate stiffeners (amber) that divide the web into panels of
length $a$ and raise the shear-buckling capacity of each panel, together
with heavier bearing stiffeners (red) over the supports that carry the
concentrated reactions into the web without crippling it.}
\label{fig:vol03ch08:plate-girder}
