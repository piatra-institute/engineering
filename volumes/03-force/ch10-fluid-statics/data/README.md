# Data for Chapter 10: Fluid statics

- `isa_profile.csv` International Standard Atmosphere pressure and
  temperature versus altitude to 32 km, with the single-scale-height
  isothermal model alongside for comparison. Generated by
  `code/standard_atmosphere.py`; plotted in `figures/fig-atmosphere.tex`.
- `fluid_properties.csv` density, specific weight, and surface tension of
  common working fluids at room temperature, used throughout the worked
  examples and exercises.
- `box_stability.csv` floating draft and metacentric height for a set of
  rectangular boxes, produced by `code/metacentre.py`, illustrating the
  beam-versus-height trade in stability.
