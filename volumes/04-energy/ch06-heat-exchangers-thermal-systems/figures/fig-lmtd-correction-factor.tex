\caption[LMTD correction factor for a one-shell-pass, two-tube-pass
exchanger]{Log-mean-temperature-difference correction factor $F$ for a
one-shell-pass, two-tube-pass exchanger, plotted against the temperature
effectiveness $P$ for three values of the capacity ratio $R$. The factor
multiplies the counter-flow log-mean difference: $\dot{Q}=UAF\,\Delta
T_{\text{LMTD,CF}}$. Each curve drops steeply as $P$ approaches the limit that a
single shell pass can reach, and the working rule is to keep $F\geq 0.75$
(dashed line); a design that lands on the steep part of the curve is fragile,
because a small error in any terminal temperature moves $F$ sharply and may push
the configuration past what it can physically achieve. A value below the cutoff
signals the need for more shell passes or a different exchanger type. The curves
here are schematic; design uses the tabulated charts in
\cite{text:incropera2007}.}
