# Data assets — Volume I, Chapter 7 (Length, area, volume, mass)

## Files

| File | Source | Used by |
|---|---|---|
| `stone_volume_three_methods.csv` | Editors' worked example: a representative stone measured by displacement, geometric-ellipsoid approximation, and smartphone photogrammetry. Values illustrative; see Section 7.5 worked example for the numerical derivation. | Section 7.5 worked example; chapter project compatibility-ratio analysis. |
| `common_material_densities.csv` | Compilation of room-temperature densities for common engineering materials, drawn from the Ashby materials charts \cite{text:ashby2017} and from the BIPM SI Brochure for water at $4 \,\si{\degreeCelsius}$ \cite{std:bipm-si2019}. Values are nominal and round; production work uses material-specific data sheets. | Section 7.2 worked example; density-identification exercises; chapter project reflection. |

## Provenance

Reference data only; not measurement results. The reader's own
measurements should replace these values when working the
exercises and project. Densities for alloys vary with composition,
heat treatment, and porosity; the table reports a representative
value and a note. For load-bearing engineering decisions, consult
the material's data sheet rather than this table.
