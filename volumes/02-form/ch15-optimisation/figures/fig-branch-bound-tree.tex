\caption[Branch-and-bound tree for the four-item knapsack]{The
branch-and-bound tree for the four-item $0/1$ knapsack of the worked
example. The root LP relaxation returns value $240$ with the
fractional variable $x_{3} = 2/3$, so the search branches on $x_{3}$.
The $x_{3} = 0$ child is integer-feasible with value $200$, which
becomes the first incumbent. The $x_{3} = 1$ child has an LP bound of
$230$, above the incumbent, so it cannot yet be pruned and branches on
its fractional $x_{2}$; both grandchildren are integer-feasible with
value $220$, raising the incumbent. Every remaining subtree has an LP
bound at or below $220$ and is pruned. Five linear-relaxation solves
settle a problem whose brute-force enumeration would test $2^{4} = 16$
leaves, and the ratio widens sharply as the number of binary variables
grows, which is why the tightness of the LP relaxation governs the
runtime of every mixed-integer solver.}
