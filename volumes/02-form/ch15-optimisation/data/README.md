# Data assets, Volume II, Chapter 15 (Optimisation)

## Files

| File | Source | Used by |
|---|---|---|
| `diet_problem.csv` | Editor's toy diet table; five foods with nominal cost and per-unit nutrient content, plus a daily requirement row. Values are illustrative and are not nutritional advice. | `code/lp_diet.py`; Section 15.3 worked LP; Simulation exercise 4 template. |
| `rosenbrock_convergence.csv` | Editor's reference convergence counts from `code/rosenbrock_methods.py` on the (-1, 1) start with the stated step sizes. Reproducible from the code. | Section 15.6; Simulation exercise 5 (target numbers). |

## Provenance

Illustrative tables only; not measurement results. The reader's own
runs should reproduce or improve on the reference numbers, and the
diet values should be replaced by reliable nutritional data
(USDA FoodData Central) for any non-toy application.
