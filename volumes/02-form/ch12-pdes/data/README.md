# Volume II Chapter 12: PDEs, supporting data

Numerical output from the chapter's reference scripts (`../code/`)
and a small reference table of thermal diffusivities cited in the
chapter prose.

| File                         | Provenance                                           |
| ---------------------------- | ---------------------------------------------------- |
| `thermal_diffusivities.csv`  | Compiled by the authors from Incropera et al. 2007.  |
| `cfl_amplification.csv`      | Computed by the authors; FTCS amplification factor.  |
| `heat_ftcs.csv`              | Generated by `../code/heat_ftcs.py`.                 |
| `wave_pulse.csv`             | Generated by `../code/wave_pulse.py`.                |
| `laplace_jacobi.csv`         | Generated by `../code/laplace_jacobi.py`.            |

The first two tables are committed at the repository so the
chapter's prose values do not depend on a successful Python run.
The latter three are populated on first run of the corresponding
script.

## Schema notes

`thermal_diffusivities.csv` columns: `material`, `alpha_m2_per_s`,
`source_pin`. The alpha values are room-temperature reference
values consistent with Table A.1 of \cite{text:incropera2007}.

`cfl_amplification.csv` columns: `r`, `k_over_kmax`, `xi`. The
amplification factor for FTCS at Courant number r and normalised
wavenumber k Delta x / pi, computed from the closed-form xi = 1
- 4 r sin^2(k Delta x / 2). The table covers r in
{0.25, 0.40, 0.50, 0.55, 0.70} across normalised wavenumbers from
0 to 1 in steps of 0.05.
