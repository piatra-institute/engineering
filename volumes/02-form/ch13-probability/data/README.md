# Data assets - Volume II, Chapter 13 (Probability theory)

Tabulated reference data and code-generated traces supporting the
chapter's worked examples and exercises.

## Files

| File | Purpose | Source |
|---|---|---|
| `distributions_reference.csv` | Recognition table of the chapter's nine canonical distributions: family, parameters, mean, variance, support, canonical mechanism. | Authors, compiled from \cite{text:ross-probability} and \cite{text:wasserman2004}. |
| `markov_trace.csv` | Running fraction of time spent in state 0 for the two-state weather chain of section 13.7, sampled at ~200 log-spaced steps up to $10^{4}$. Generated by `../code/markov_chain_simulation.py`. | Computed by the authors via the Markov-chain simulation script. |
| `clt_histograms.csv` | Histograms of the standardised sample mean of $\mathrm{Exp}(1)$ at $n = 1, 5, 30, 100$, $10\,000$ trials each, $40$ bins on $[-4, 4]$. Generated by `../code/clt_demo.py`. | Computed by the authors via the CLT-demo script. |

## Regenerating

Both `markov_trace.csv` and `clt_histograms.csv` are reproducible by
running their generating scripts with the seeds named in the script
files (`2026` by default). The reference table is hand-curated.
