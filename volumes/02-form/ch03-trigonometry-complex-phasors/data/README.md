# Data assets - Volume II, Chapter 3 (Trigonometry, complex numbers, phasors)

Reference data for the chapter's worked examples and exercises.

## Files

| File | Purpose | Provenance |
|---|---|---|
| `rlc_impedance.csv` | Series RLC impedance magnitude and phase versus frequency for the worked example (R = 100 ohm, L = 100 mH, C = 10 uF), 200 points logarithmically spaced over 10 Hz to 10 kHz. | Computed by `code/rlc_response.py`. |
| `canonical_angles.csv` | The five canonical first-quadrant angles (0, pi/6, pi/4, pi/3, pi/2) with exact and decimal values of sine, cosine, and tangent. | Computed from the unit-circle definitions of section 3.1. |

## Provenance

The `rlc_impedance.csv` file is regenerated by running
`uv run rlc_response.py` from the `code/` directory.
The `canonical_angles.csv` file is an editorial reference
table; the values are exact and follow from the equilateral and
isoceles-right-triangle constructions of section 3.1.
