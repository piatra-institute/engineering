# Data assets - Volume II, Chapter 5 (Differentiation)

## Files

| File | Purpose | Used by |
|---|---|---|
| `sample_noisy_sinusoid.csv` | A 41-sample noisy sinusoid $y(t) = \sin(2\pi t) + \varepsilon(t)$, $\varepsilon \sim \mathcal{N}(0, \sigma^{2})$ with $\sigma \approx 0.04$, sampled at $\SI{40}{\hertz}$ over one period. Three columns: $t$ in seconds, the noisy measurement, and a reference column carrying the noise-free analytical value at the same time grid. The reference column lets the reader compute the realised error of a derivative estimator without re-running the noise generator. | Section 5.7 (Differentiating noisy data); the chapter Project (noisy-signal differentiation) when the reader does not wish to generate their own data; the Simulation exercise on Savitzky-Golay smoothing. |

## Provenance

Synthetic. Generated by the editor with seed 2026 against the closed
form $y_{\text{true}}(t) = \sin(2\pi t)$, with $\varepsilon \sim
\mathcal{N}(0, \sigma^{2})$ added per sample. Values rounded to three
decimal places; the reader who needs more precision should regenerate
from the closed form. The reference column is the noise-free value at
the same time grid, included as a verification target for derivative
estimators.

## Conventions

- All time values in SI seconds; signal values dimensionless.
- Header row present; comma-separated; UTF-8 encoded.
- One header row, then 41 data rows for one full cycle plus the
  closing sample at $t = 1.000 \,\si{\second}$.
