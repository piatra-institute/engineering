# Data assets — Volume II, Chapter 18 (Mathematical modelling)

Numerical data sets used by the chapter's worked examples and
exercises. Each file is plain CSV with a header row; provenance and
acquisition conditions are stated below.

## Files

| File | Provenance | Used by |
|---|---|---|
| `cooling_cup.csv` | Authors' measurement, current as of 2026-05. Tap water in a 300 mL ceramic mug, initial temperature 90$^\circ$C, room temperature 22$^\circ$C, ambient still air, thermometer accuracy $\pm 0.5\degC$ per Volume I Chapter 3 calibration. Seven readings at five-minute intervals. | Worked example 18.6; Exercises 18.11, 18.12; `cooling_fit.py`, `bootstrap_interval.py`, `cross_validation.py`. |
| `logistic_growth.csv` | Synthetic data generated from the logistic equation with $r = 0.7\,\si{\per\hour}$ and $K = 2.0$ (millions per mL), with Gaussian noise $\sigma = 0.05$ added. Generated by the project for pedagogical use; not a real microbiological measurement. Seed: `numpy.random.default_rng(2026)`. | Worked example 18.7 (biological system); Exercise 18.14; `logistic_fit.py`. |
| `pendulum_periods.csv` | Authors' measurement, current as of 2026-05. Brass-bob pendulum on a low-friction pivot, five lengths from 0.50 to 1.50 m measured with a tape (accuracy $\pm 2$ mm), period averaged over ten swings with stopwatch (per-swing uncertainty $\pm 0.05$ s, so per-period-average uncertainty $\approx 0.016$ s). | Exercise 18.13; reference for the second estimation block. |
| `beam_deflection.csv` | Computed by the authors from `sensitivity_propagation.py`'s nominal beam (W14x22 steel, $L = 5$ m, $w = 2$ kN/m). Each row: a perturbed parameter value and the corresponding $y_{\max}$. Provided so a reader can re-derive the tornado diagram independently. | Exercise 18.5; Exercise 18.17; `sensitivity_propagation.py`; figure 18.6. |

## Format

All files use UTF-8, Unix line endings, commas as separators, and a
single header row naming each column with its units. Numerical values
use the period as decimal point and no thousands separator.

## A note on the cooling-cup acquisition

The seven readings are mean values; each was read three times and the
median was recorded. The mug was placed on a wooden surface in still
ambient air (no draught, no direct sunlight) so that the convective
heat-transfer coefficient could be approximated as constant. The
thermometer was held in the centre of the liquid volume, away from
the mug's wall, so that the reading approximates the bulk
temperature rather than the wall temperature. The acquisition
conditions match the cooling-cup model's regime of applicability.
