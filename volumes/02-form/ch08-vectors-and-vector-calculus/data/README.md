# Data for Vol II Ch 8 (Vectors and vector calculus)

Reference meshes and a precomputed sweep output for the divergence-theorem volume calculation.

## Files

| File | Description |
|---|---|
| `unit_cube.obj` | Unit cube, 8 vertices and 12 triangles, with consistent outward orientation. |
| `tetrahedron.obj` | Regular tetrahedron inscribed in the unit cube, volume $1/3$. |
| `sweep_reference.csv` | Output of `../code/triple_product_sweep.py` showing the second-order convergence of the divergence-theorem volume on a geodesic icosphere as the mesh is refined. |

## Provenance

The cube and tetrahedron meshes were authored by the editorial team for Vol II Ch 8 (current as of 2026) and verified against the analytical volumes by `triple_product_sweep.py`. The icosphere meshes are generated by the script at run time and are not stored on disk.

The CSV is committed for cross-reference: a reader running the sweep should reproduce the volume column to floating-point precision on the closed-form cases and to four significant figures on the icosphere column at refinement four.
