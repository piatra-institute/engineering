% Input resistance of a finite R-R ladder converging to the semi-infinite
% value as the number of sections grows. Each section adds a series R and a
% shunt R; the finite-ladder resistance R_N obeys the fixed-point recursion

% fixed point is the golden value R* = (1+sqrt5)/2 = 1.618. The plot shows
% R_N approaching R* from alternating sides, the cobweb of a contracting map.
% Values (R=1 ohm): R0=1, R1=1.5, R2=1.6, R3=1.6154, R4=1.6176, R5=1.61797,
% R*=1.618034.

\draw[->, thick] (0, -0.2) -- (0, 4.4) node[above] {input resistance $R_{N}/R$};

\caption[Convergence of the ladder-network input resistance]{Input
resistance of a finite resistive ladder against the number of sections
$N$, normalised to the per-element resistance $R$. Adding one section maps
the previous input resistance $R_{N-1}$ through
$R_{N}=R+R\,R_{N-1}/(R+R_{N-1})$, a contracting fixed-point iteration whose
limit is the semi-infinite ladder resistance $R^{*}=\phi R$ with
$\phi=(1+\sqrt5)/2$. The iterates approach $R^{*}$ within one part in a
thousand by the fourth section, so a physical ladder is indistinguishable
from the infinite one after a handful of stages. The limit is the positive
root of $R^{*}=R+R\,R^{*}/(R+R^{*})$, the same golden ratio the
generating-function case study read off a pole, here read off a fixed
point.}
