# Data assets - Volume II, Chapter 2 (Functions and graphs)

Small reference datasets used by the chapter's figures, code scripts,
and exercises. Each file is small enough to ship with the chapter
source; large outputs of the simulation scripts are written to this
directory at run time and are not committed.

## Files

| File | Source | Purpose |
|---|---|---|
| `thermistor_calibration.csv` | Authors' tabulation from the Steinhart approximation with $B = 3800\,\si{\kelvin}$ and $R_{25} = 10\,\si{\kilo\ohm}$, a typical 10K NTC bead. Eight points across $0\,\degC$ to $80\,\degC$. | Drives the thermistor-fit figure (fig:vol02:ch02:thermistor-fit) and the design exercise. Cross-validation target for `thermistor_calibration.py`. |
| `challenger_oring_flights.csv` | Tabulated from the Rogers Commission report, Volume I, Figure 7 (`acc:nasa-challenger`). Twenty-four pre-STS-51-L Shuttle flights with launch ambient temperature and observed O-ring damage index. | Drives the Challenger chart figure (fig:vol02:ch02:challenger-chart) and the Failure-analysis exercises. |
| `rc_discharge_decay.csv` | Authors' tabulation of $V(t) = V_{0} \exp(-t/\tau)$ for $V_{0} = 4.00\,\si{\volt}$, $\tau = 1.50\,\si{\second}$, sampled at 50 points across $t \in [0, 5]\,\si{\second}$. | Drives the semilog-decay figure (fig:vol02:ch02:semilog-decay) and the Simulation exercises. |

## Provenance

`thermistor_calibration.csv`: each row is computed from
$R(T) = R_{25} \exp[B(1/T - 1/T_{25})]$ with $T$ in kelvin,
$R_{25} = 10\,000\,\Omega$, $T_{25} = 298.15\,\si{\kelvin}$, and
$B = 3800\,\si{\kelvin}$. The values are quoted to the nearest ohm,
matching the resolution of a four-digit benchtop ohmmeter.

`challenger_oring_flights.csv`: replotted from the Rogers Commission
report Volume I Figure 7 (Shuttle solid rocket motor field-joint
O-ring damage history through STS-61-C, i.e., the twenty-four
Shuttle flights preceding STS-51-L). The damage index is the integer
tally of erosion and blow-by incidents at field and nozzle joints.
The launch ambient temperatures are recorded in the report to the
nearest degree Fahrenheit and converted here to degrees Celsius.

`rc_discharge_decay.csv`: synthetic, generated by evaluating the
closed-form decay at the stated sample times. The CSV is used in
the chapter as a clean comparison curve for the noisy-fit
simulation exercise (`semilog_fit.py`).
