\caption{A rooted tree on eight vertices. The root $r$ has three
children; vertex $c$ has one child $f$, which has one child $g$. The
tree has $8$ vertices and $7$ edges, in agreement with the
$n - 1$ edge count for any tree. Leaves are $d, e, b, g$; internal
vertices are $r, a, c, f$. The height is $3$, the length of the
longest root-to-leaf path ($r \to c \to f \to g$). A balanced tree
on $n$ vertices has height $\Theta(\log n)$; the degenerate chain
$r \to c \to f \to g$ illustrates the unbalanced case whose height
is $\Theta(n)$, the worst case a self-balancing structure exists
to prevent.}
