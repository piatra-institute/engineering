\caption{A Karnaugh map for $f(a,b,c)$ true on the minterms
$\{001,011,110,111\}$. Columns run in Gray-code order $00,01,11,10$
so that horizontally or vertically adjacent cells differ in exactly
one variable, and a rectangular block of $2^{k}$ ones collapses to a
single product term with $k$ fewer literals. The blue block covers
$\lnot a\wedge c$ and the red block covers $a\wedge b$, giving the
minimised cover $f=(\lnot a\wedge c)\vee(a\wedge b)$, two product
terms in place of four. The map is a spatial layout of the Boolean
identity $(X\wedge y)\vee(X\wedge\lnot y)=X$: what the eye reads as an
adjacent pair, the algebra reads as a cancelled variable.}
\label{fig:vol02:ch17:karnaugh}
