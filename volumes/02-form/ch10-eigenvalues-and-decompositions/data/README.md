# Data assets — Volume II, Chapter 10 (Eigenvalues and decompositions)

Reference datasets supporting the chapter's exercises, worked
examples, and project.

## Files

| File | Source | Year | Used by |
|---|---|---|---|
| `synth_pca_dataset.csv` | Synthesised by the project editor with a planted rank-2 structure on five features plus small noise. | 2026 | Chapter project (PCA from scratch), as a debugging dataset where the principal directions are known a priori. |
| `three_mass_modal.csv` | Computed by the editor from the closed-form modal solution of three equal masses, four equal springs of stiffness $k$, between walls. | 2026 | Section 10.8 modal analysis worked example; Diagnosis exercise on near-degenerate modes. |

## Provenance

`synth_pca_dataset.csv` was generated by drawing latent factor scores
from $\mathcal{N}(0, 1)$, projecting through a random orthonormal
$5 \times 2$ loading matrix scaled by $(10, 2)$, and adding isotropic
Gaussian noise with standard deviation $0.05$. The recipe is in
`code/pca_three_routes.py` (`synth_data` function); the file lets the
reader run the PCA pipeline against a dataset whose true principal
directions are known.

`three_mass_modal.csv` contains the analytical mode shapes and
eigenvalues of the stiffness matrix
$\mat{K} = k \begin{pmatrix} 2 & -1 & 0 \\ -1 & 2 & -1 \\ 0 & -1 & 2
\end{pmatrix}$, normalised so that each mode shape has unit Euclidean
norm. Computed from $\lambda_{i} = 2 - 2\cos(i\pi/(n+1))$ for $n=3$
masses and $i = 1, 2, 3$.

## Use

Files are CSV (UTF-8, RFC 4180) with a header row. Numerical fields
are quoted to six decimal digits, more than enough for the chapter's
worked examples. The reader can substitute their own datasets while
preserving the file structure; the project encourages this and treats
the synthesised file as a debugging starting point only.
