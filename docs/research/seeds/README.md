# Research seeds

This directory holds the project's seed-stage research notes. A seed is a captured thinking pass: an idea, framing, or quotable sentence from external reading, distilled into a self-contained file after the source has been set aside. Seeds are neither finished research nor canon; they are the audit trail of the conceptual moves that feed the book.

The lane's discipline is documented at:

- `SCHEMA.md`: the format every seed takes.
- `../../voice.md`: voice rules apply to seeds as to chapter prose.
- `../../citation-policy.md`: bibliography-prefix conventions used inside seed frontmatter.

Seeds are not cited from chapter prose. When a seed makes it into the book, the chapter cites the underlying source; the seed stays in place as genealogy.
